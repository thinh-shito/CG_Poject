Dưới đây là hướng dẫn để chạy đồ án
Yêu cầu: cài đặt Visual studio code và cài extension Live server
Bước 1: Mở folder chứ source code bằng vscode
Bước 2: Mở file index.html
Bước 3: Chuột phải và chọn Open with Live server
Bước 4: Play game

Lưu ý: nếu xảy ra các trường hợp:
	* Không hiển thị texture trên máy bay.
	* Level thay đổi nhưng background thì không.
	-> Xin refresh lại page.
	Nếu đã refresh mà vẫn vậy thì có thể chọn một trong 2 trình duyệt sau để thực hiện (Firefox, Safari).
